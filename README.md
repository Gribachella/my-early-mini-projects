# number_guessing_game
Привет! Это мой первый мини-проект - "Игра в числовую угадайку", который я написал
на питоне. Программой загадываеться число, вами - угадываеться. Если попадаете на меньшее,
программа выведет соответствующую строку в терминале, тоже верно и для ввода большего числа.

Для того чтобы начать играть нужно скачать питон и запустить .py файл

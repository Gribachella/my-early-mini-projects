# number_guessing_game
Привет! Это мой первый мини-проект - "Игра в числовую угадайку", который я написал
на питоне. Программой загадываеться число, вами - угадываеться. Если попадаете на меньшее или большее нежели,
загаданное, то программа выведет соответствующую строку в терминале.

Для того чтобы начать играть нужно скачать питон и запустить .py файл

В игре приличная валидация ввода: проверяются тип данных
Так что независимо от чуши, которую может вводить 
пользователь, программа все равно не упадет и продолжит работать.

Игра получилось довольно живенькой благодаря случайному выбору строк, которые программа возвращает уже в 
основном цикле игры при победе, слишком долгой победе и других сценариях. Также есть небольшая пасхалка.
